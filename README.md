# gym-trainer — зал-игра, сайт тренера

Промо-сайт персонального тренера (силовые + питание) как маленькая 3д-игра:
посетитель управляет качком-персонажем, ходит по залу, взаимодействует со станциями.
Каждая станция — раздел сайта. Прогресс исследования «изучено N/8» ведёт к CTA записи.

## Запуск

```
python3 -m http.server 8000
# → http://localhost:8000
```

Без сборки. three.js завендорен в `vendor/`.

Debug: `?at=rack|bench|dumbbells|board|bag|cardio|kitchen|reception` — заспавниться у станции.
Проверка визуала — puppeteer-core со системным Chrome (голый `chrome --headless --screenshot`
с `--virtual-time-budget` не дожидается rAF-анимаций — не использовать).

## Управление

- Десктоп: WASD/стрелки — ходьба, мышь (драг) — камера, E — действие, Esc/E — закрыть панель.
- Мобилка: динамический джойстик под пальцем (левая половина), драг справа — камера,
  круглая кнопка действия. Онбординг как в играх, исчезает при первом движении.

## Механики

- **Персонаж** — низкополигональный качок с утрированным силуэтом (Mashle-нота: плечи,
  предплечья, уверенность). Процедурные анимации: походка, присед со штангой, удар по мешку.
  Козырёк кепки и латунная бляха на груди = индикатор «где перёд».
- **Упражнения на всех снарядах** (единый режим exercise): присед на помосте
  (двухзвенная кинематика ног — стопы плоско на полу), жим лёжа (лежит на скамье,
  гриф в руках), попеременные сгибания с гантелями у стеллажа, бег на дорожке,
  сидение на лавке. Во время упражнения коллайдеры персонажа не выталкивают.
- **Мешок** (E): руки бьют по очереди, удержание E / кнопки — серия без остановки;
  подшагивание на дистанцию удара; после третьего удара панель открывается,
  но E продолжает бить — закрытие через Esc/тап по футеру/отойти.
- **Хореография фокуса** (система, не эффект): E → свет ведёт (зал мягко темнеет,
  тёплый луч на станцию) → камера доворачивается и приближается, объект уходит
  в свободную от панели часть кадра (влево на десктопе, вверх на мобилке) →
  карточка входит с задержкой 280мс, контент поднимается каскадом. Закрытие
  разматывает всё обратно.
- **Панели** — игровое меню, крупное и брутальное: латунная рамка со срезами и
  уголками-скобами, зелёная плашка зоны с номером, «статы», нумерованные шаги,
  футер с прогрессом. Виньетка по краям кадра.
- **Прогресс** — HUD «изучено N/10»; на 10/10 — финальный тост с CTA. Станции:
  помост, жим, гантели, мешок, кардио, лавка (мотивация с остринкой), полка славы
  («рыбный» контент о тренере: кубки, медали, фото-плейсхолдер), бар, прайс-доска, ресепшн.
- **Типографика** — Oswald (вариативный, OFL, завендорен в `vendor/fonts/`, кириллица),
  и в UI, и в canvas-текстурах мира; шрифт грузится до постройки сцены.
- **Окружение** — дневной зал: светлый бетон, дерево, глубокий зелёный, латунь
  (сознательно не «подвальная качалка» — ЦА взрослая). Входная зона: угловой
  L-ресепшн с латунной полосой и звонком + прайс-доска рядом на стене + бар
  (стойка, стулья, полки с бутылками, вывеска «БАР»). Постеры-сторителлинг, в том числе
  два плаката качков с силуэтом «двойной бицепс» («Старая школа», «День ног — святое»).
  Часы, растения, шведская стенка, латунная разметка зон, тёплый контровой свет.

## Архитектура

```
js/world.js      палитра, комната, оборудование, коллайдеры, постеры
js/character.js  риг из примитивов + позы (idle/walk/squat/punch)
js/controls.js   WASD + джойстик + камера-следование + коллизии
js/ui.js         онбординг, промпт, FAB, панель, прогресс
js/main.js       сцена, станции, режимы (free/panel/squat), цикл
```

## Статус: четвёртая итерация (все снаряды, хореография фокуса, Oswald)

Заглушки: тексты (тон согласовать), цены (`···`), имя, ссылка на телеграм.
Типографика временная. Известные упрощения: камера может входить в высокие объекты,
у персонажа нет глаз/лица, звука нет.

## Идеи на следующие итерации

- Больше упражнений: жим на скамье (лечь + жать), подтягивания на шведской стенке.
- Дорожка: персонаж бежит на месте, пока открыта панель кардио.
- Секундная стрелка часов = реальное время; свет меняется по времени суток посетителя.
- Пересобрать зал по фото реального зала тренера.
- Звук: лязг, шаги, глухой удар мешка (с выключателем).
- Лицо/глаза персонажу; возможность «выбрать себя» (пол/тип) при входе.
